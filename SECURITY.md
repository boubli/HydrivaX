# Security Policy

## Supported Versions

The following versions of HydrivaX receive security updates:

| Version | Supported |
| ------- | --------- |
| v2.x    | ✅ Yes     |
| v1.x    | ✅ Yes     |
| < v1.0  | ❌ No      |

## Reporting a Vulnerability

If you find a security vulnerability in HydrivaX, please do **not** open a public issue. Instead, report it privately to the maintainer:

- **Email**: [bbb.vloger@gmail.com](mailto:bbb.vloger@gmail.com)

Please include as much information as possible, including:
- A description of the vulnerability.
- Steps to reproduce (a proof-of-concept if possible).
- The potential impact.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible. Thank you for helping keep HydrivaX secure!
