Why HydrivaX

Everything a container doesn't need is gone. What remains is a fast, headless foundation built for deploying your own services on demand.

⚡

Aggressively stripped

No kernel modules, no firmware, no bootloader, no locales. The rootfs is trimmed to what an LXC container actually needs.

📦

On-demand modules

Nothing is pre-installed. Run hx-deploy docker or any of 99 modules when you need them — remove them just as easily.

🖥️

Headless CLI first

Built around hx-deploy, hx-status, and hx-info. A terminal-native experience for infrastructure operators.
