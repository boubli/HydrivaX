Quick Start Guide

Deploy HydrivaX in under 5 minutes on Proxmox VE.

1

Prerequisites

Ensure you have a running Proxmox VE host and SSH or console access to it. You do not need to download the ISO or template manually; our Autopilot script handles it.

2

Run the Autopilot Installer

Log in to your Proxmox VE node's shell and execute the following command. It will download the HydrivaX template and automatically provision an LXC container for you.

root@pve:~# bash -c "$(curl -fsSL https://boubli.github.io/HydrivaX/install-lxc.sh)"

Follow the interactive prompts to assign an ID, hostname, and password.

3

First Boot & Verification

Once the container is running, log into it using the password you set. Verify the system is healthy using the built-in status command.

root@hydrivax:~# hx-status
4

Deploy Your First Module

HydrivaX ships stripped down. To start running services, deploy a module like Docker. This command automatically installs dependencies, configures the daemon, and starts the service.

root@hydrivax:~# hx-deploy docker

Congratulations! You have successfully deployed HydrivaX and provisioned your first service module.
