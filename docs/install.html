Install HydrivaX OS v2.0

Server ISO for CLI infrastructure, Desktop ISO for a graphical HydrivaX environment, and LXC for Proxmox containers.

Server ISO: VM or Bare Metal

HydrivaX-v2.0-Server-amd64.iso is install-first media. Pick Install HydrivaX OS v2.0 to Disk from the GRUB menu, or log in as root and run hx install.

1

Download and verify

# Windows Get-FileHash HydrivaX-v2.0-Server-amd64.iso -Algorithm SHA256 # Linux / macOS sha256sum HydrivaX-v2.0-Server-amd64.iso

Expected SHA256: 36eb69ac8f5ffb78323abe7c0732ac584228d439ade3c46351ed6adc10127ecb

2

Boot the ISO

Use Proxmox VM, VirtualBox, VMware, or a flashed USB drive for physical machines. For Proxmox, upload the ISO:

$ scp HydrivaX-v2.0-Server-amd64.iso root@PROXMOX_IP:/var/lib/vz/template/iso/

Recommended VM settings: BIOS/SeaBIOS, 1-2 vCPU, 1-2 GB RAM, 10-20 GB disk, SATA/SCSI/VirtIO disk, NAT or bridged network.

At the GRUB menu, select Install HydrivaX OS v2.0 to Disk (hotkey i). For Proxmox serial console, use Install HydrivaX OS v2.0 to Disk - Serial. The native installer starts immediately on tty1.

3

Log in to HydrivaX

user root pass hydrivax
4

Install permanently to disk

The installer wipes the selected disk, creates a universal GPT layout with BIOS boot, EFI, and ext4 root partitions, clones the HydrivaX runtime with rsync -aAXv, writes UUID-based /etc/fstab, and installs GRUB for BIOS and UEFI.

$ hx install # direct commands also work sudo hx-deploy install sudo hx-install

Examples of valid target disks: /dev/sda, /dev/vda, /dev/nvme0n1, /dev/mmcblk0.

5

Reboot into installed OS

When the installer finishes, remove the ISO/USB media and reboot. The machine will boot from the installed disk.

Desktop ISO: HydrivaX GUI

HydrivaX-v2.0-Desktop-amd64.iso includes HydrivaX Desktop Core. Use Install HydrivaX OS v2.0 to Disk in the GRUB menu for a headless install, or use the try mode to explore the HydrivaX desktop.

1

Download and verify

# Windows Get-FileHash HydrivaX-v2.0-Desktop-amd64.iso -Algorithm SHA256 # Linux / macOS sha256sum HydrivaX-v2.0-Desktop-amd64.iso

Expected SHA256: c15b96052e5dede0879c6373b6cfffd25f754ffceceba466960b73b8d1635c56

2

Install from the GRUB menu (recommended)

Select Install HydrivaX OS v2.0 to Disk at boot. This skips the desktop and runs the same text installer used on the Server ISO.

3

Or boot to HydrivaX Desktop

Choose Try HydrivaX OS v2.0 without installing - VGA Text to open the secondary desktop/rescue mode. A terminal installer opens automatically there too.

4

Install from the desktop

Use the automatic installer window, or click Install HydrivaX OS on the desktop/dock. It runs the same production installer:

$ hx install
5

Explore the GUI

The HydrivaX desktop includes a HydrivaX wallpaper, custom dock icons, HydrivaX apps, terminal fastfetch output, browser, settings, and file manager.

New v2.0 Commands

These commands are available inside the ISO images.

hx-deploy install

Branded entry point for installing HydrivaX OS permanently to a VM or physical disk.

hx-install

Native disk installer. Handles SATA/SCSI/VirtIO, NVMe, eMMC, universal GPT partitioning, ext4 formatting, rsync clone, UUID fstab, and BIOS/UEFI GRUB.

hydrivax-install

Compatibility alias for users who remember the older installer command.

Proxmox LXC Template

LXC remains a lightweight Proxmox container template. It is not for bare-metal disk installation and does not include the GUI installer workflow.

1

Upload the template

$ scp HydrivaX-v2.0-LXC.tar.zst root@PROXMOX_IP:/var/lib/vz/template/cache/
2

Create a container

Use the Proxmox UI to create a CT from the template. Then SSH in and use the CLI tools.

$ hx-info $ hx-status $ hx-deploy list
