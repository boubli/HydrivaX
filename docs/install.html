Install on Proxmox

Download the template, upload it to your Proxmox host, create a container, and start deploying modules.

1

Download the template

Grab HydrivaX-v2.0-LXC.tar.zst from the Releases page.

2

Upload to Proxmox

Copy the archive to your Proxmox template cache directory.

$ scp HydrivaX-v2.0-LXC.tar.zst root@PROXMOX_IP:/var/lib/vz/template/cache/
3

Create a container

Open the Proxmox web UI, go to your node → CT Templates, and create a new LXC container from the HydrivaX template. Configure CPU, RAM, and storage as needed.

4

First commands

SSH into your new container and explore the CLI toolkit.

$ hx-info $ hx-deploy list $ hx-status $ hx-deploy docker

What to expect

On first login you'll see the HydrivaX MOTD banner. The system is intentionally minimal — install only what you need.

root@hydrivax:~
