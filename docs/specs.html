Specifications

HydrivaX v2.0 ships in three clean targets: Server ISO, Desktop ISO, and Proxmox LXC.

HydrivaX OS v2.0 Server

Installable live server ISO for VMs and bare metal.

Base OSDebian 12 (Bookworm) Live
ISO size574 MB
FormatBootable amd64 ISO
EditionHeadless CLI server
Install commandsudo hx-deploy install or sudo hx-install
Partition layoutMBR/msdos + single ext4 root partition
BootloaderGRUB installed to target disk
Live userlive / hydrivax
TargetProxmox VM, VirtualBox, VMware, mini PCs, old desktops, bare metal

HydrivaX OS v2.0 Desktop

Graphical HydrivaX Desktop Core ISO with first-boot installer.

Base OSDebian 12 (Bookworm) Live
ISO size1020 MB
DesktopXFCE + HydrivaX Desktop Core custom UI
LoginLive desktop autologin
Installer UXInstaller launches automatically on first desktop boot
Included UIHydrivaX wallpaper, dock, branded app launchers, terminal fastfetch output
Install commandsudo hx-install
TargetVMs, laptops, mini PCs, desktop PCs, bare metal

HydrivaX v2.0 LXC Template

Proxmox container template, kept CLI-only and lightweight.

Base OSDebian 12 (Bookworm)
Template size99 MB
Rootfs size346 MB
FormatProxmox LXC .tar.zst
EditionHeadless CLI container
Disk installerNot included; LXC is container-only
Target platformProxmox VE containers
