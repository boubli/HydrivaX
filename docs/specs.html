What's stripped (LXC)

The LXC template removes everything a container doesn't need.

No kernel modules

Kernel modules are not needed inside an LXC container and are removed to save space.

No firmware or bootloader

Firmware blobs and bootloader packages are purged from the rootfs.

No locales bloat

Unnecessary locale data is stripped. The system stays lean and fast to deploy.
